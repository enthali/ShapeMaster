ShapeMaster PowerPoint Add-in

ShapeMaster is an open source PowerPoint add-in that enhances shape manipulation, making it easy to resize, swap, and annotate shapes in your presentations.

How to Install

There are two ways to install ShapeMaster. Please read carefully and choose the option that fits your needs and experience level:

  1. Developer/Secure Way:
    For developers or security-conscious users.
    View Developer Instructions
  2. Trusting User Way:
    For users who want a quick install and are willing to trust the provided files and certificate.
    View User Installation Guide

Pros, Cons & Security Warning

  • Developer Way: Most secure, but requires technical skills and tools.
  • Trusting User Way: Easier, but requires trusting this website and the certificate.
    Never trust a webserver blindly!
