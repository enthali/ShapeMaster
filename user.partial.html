User Installation (Trusting Way)

This method is for users who want a quick install and are willing to trust the provided certificate and setup file.

Installation Steps

  1. Download the publisher certificate:
    Download Certificate
  2. Double-click the ShapeMaster_Public.cer file.
  3. Click “Install Certificate…”.
  4. Choose “Local Machine” (or “Current User” if you don’t have admin rights).
  5. Select “Place all certificates in the following store.”
  6. Click Browse and select “Trusted Root Certification Authorities.”
  7. Complete the wizard and confirm any security prompts.
  8. Download and run the installer:
    Download ShapeMaster setup.exe
  9. Follow the prompts to complete installation.
  10. Open PowerPoint and look for the Shape Master tab in the ribbon.

After installing the certificate, try running the installer again. The error should be resolved, and the add-in should install without trust warnings.

Pros

  • Quick and easy installation.
  • No development tools required.

Cons & Security Warning

  • You must trust this website and the files provided.
  • Installing a certificate gives the publisher significant trust on your system.
  • Never trust a webserver blindly. Only proceed if you understand the risks.
  • Remove the certificate if you no longer use the add-in.
