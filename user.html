User Installation (Trusting Way)

This method is for users who want a quick install and are willing to trust the provided certificate and setup file.

Steps

  1. Download and install the publisher certificate:
    Download Certificate
    Double-click the certificate and install it to the Trusted Root Certification Authorities store for your user account.
  2. Download and run the installer:
    Download ShapeMaster setup.exe
  3. Follow the prompts to complete installation.
  4. Open PowerPoint and look for the Shape Master tab in the ribbon.

Pros

  • Quick and easy installation.
  • No development tools required.

Cons & Security Warning

  • You must trust this website and the files provided.
  • Installing a certificate gives the publisher significant trust on your system.
  • Never trust a webserver blindly. Only proceed if you understand the risks.
  • Remove the certificate if you no longer use the add-in.

Warning: Only install software and certificates from sources you trust. If in doubt, use the developer method instead.
