Developer/Secure Installation

This method is for developers or security-conscious users who want to review and build the add-in from source.

Impressum

Steps

  1. Clone the repository:
    git clone https://github.com/enthali/ShapeMaster.git
  2. Review the source code to ensure it meets your security standards.
  3. Follow the README instructions to build the project.
  4. Install the add-in using your built files.

Pros

  • Maximum security and transparency.
  • Ability to audit and modify the code.

Cons

  • Requires development tools and experience.
  • More time-consuming than the user installation.
